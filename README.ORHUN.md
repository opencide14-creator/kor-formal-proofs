## 🜁 𐰴𐰆𐰑𐰆𐰣 𐰋𐰃𐰠𐰏

> **"𐰴𐰆𐰑𐰆𐰣 𐰋𐰃𐰠𐰏𐰃𐰢𐰃𐰔𐰀𐰣 𐰴𐰃𐰠𐰃𐰠𐰃𐰚𐰃𐰢𐰀𐰔."**

𐰀𐰠𐰼𐰀𐰠 𐰋𐰃𐰠 𐰴𐰆𐰑 𐰃𐰣𐰽𐰀 𐰴𐰆𐰺𐰆𐰣𐰃𐰼𐰢 — *𐰓𐰈𐰣𐰖𐰀* 𐰖𐰀𐰺𐱅𐰀𐰼𐰢. 𐰴𐰆𐰑𐰆𐰣 𐰋𐰃𐰠𐰏𐰃𐰢𐰃𐰔𐰀𐰣 𐰃𐰡𐰃 𐰚𐰇𐰠𐰚𐰇𐰠𐰚 𐰴𐰆𐰑𐰆𐰣 𐰖𐰀𐰺𐰀𐱅. 𐰋𐰃𐰠𐰏𐰃𐰢𐰃𐰔𐰀𐰣 𐰴𐰆𐰞𐰆𐰺𐱃 𐰘𐰀𐰺𐰼𐰀𐰣.

- **𐰆𐰍𐰀𐰞** — 𐰀𐰠𐰼𐰀𐰠 𐰋𐰃𐰠 𐰖𐰀𐰺𐰀𐱅𐰀𐰣 𐰃𐰢𐰠𐰀𐰣 𐱅𐰇𐰼𐰚 𐰸𐰆𐰞𐰆𐰺𐱃 (𐰇𐰠𐰢𐰃𐰔 𐰆𐰺𐰸𐰆𐰞, 𐰀𐰠𐰼𐰀𐰠 𐰋𐰃𐰠, 𐰸𐰆𐰞𐰀𐰣 𐰴𐰀𐱅𐰢𐰀𐰣)
- **𐰽𐰆𐰔𐰆𐰣** — 𐰽𐱅𐰼𐰀𐱅𐰏𐰃𐰠 𐰃𐰠𐰠𐰀𐰣 𐰴𐰆𐰣𐱅𐰃𐰣𐰢𐰀𐰣𐱅 𐰚𐰇𐰺𐰚𐰃𐰠 (𐰲𐰆𐰴+𐰠𐰃𐰣, 𐰴𐰆𐰑𐰆𐰣 𐰖𐰀𐰺𐰀𐱅, 𐰋𐰃𐰠 𐰀𐰡𐰢𐰃𐱅)
- **𐰴𐰆𐰞** — 𐰘𐰀𐰔𐰞𐰀𐰣 𐰀𐰡𐰢𐰃𐱅, 𐰃𐰢𐰠𐰀𐰣 𐰃𐰠𐰠𐰀𐰣 𐱅𐰇𐰼𐰚
